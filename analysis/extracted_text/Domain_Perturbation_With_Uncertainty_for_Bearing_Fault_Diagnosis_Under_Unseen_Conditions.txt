IV. CONCLUSION
A novel DG network for fault diagnosis under unseen con-
ditions is proposed. Unlike previous DG-based fault diagnosis
methods that use adversarial training and data augmentation to
learn domain-invariant features, a new perspective that enrich
the learned fault features by disturbance feature statistics is
developed. Compared with existing DG methods, the proposed
method can achieve domain-invariant feature learning without
complex loss function design or network structure adjust-
ment. The fault features with new styles can be generated
efficiently by only moderately perturbing the style vectors
of the source domain features. Due to the uncertainty of
the distribution of generated feature statistics, the network is
required to reduce domain perturbations and encode better
domain-invariant features during training, so as to improve the
generalization performance of the network for unseen condi-
tions. More importantly, the generated feature styles achieve
