# Demo Reference Policy

Bundled preview images and existing demo outputs are not an authority for this skill. Do not read image or PDF assets and do not adapt visual style from stored previews.

If a user asks to reproduce a prior demo, require the numerical data or the original plotting source outside `assets/`. Then regenerate the figure through the active IEEE Python template.

Allowed use:

- Reuse the concept of a chart family, such as ablation bar, domain radar, or feature embedding.
- Reimplement the plot from user-supplied arrays.
- Apply the IEEE contract for dimensions, fonts, LaTeX variables, color, uncertainty, and export.

Disallowed use:

- Inspecting image/PDF files in `assets/`.
- Treating old preview appearance as a target style.
- Copying hidden data from bundled scripts under `assets/`.
