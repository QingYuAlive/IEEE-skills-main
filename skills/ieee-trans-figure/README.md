# ieee-trans-figure

This skill generates IEEE Transactions-ready Python plotting code for TCYB, THMS, and TNNLS submissions.

It enforces exact column widths, permitted fonts, LaTeX math rendering, high-contrast colorblind-aware palettes, uncertainty reporting, OOD protocol visibility, ablation defense, and reproducible PDF/PNG export.

The skill is Python-only and must not inspect or modify image/PDF files under `assets/`.
