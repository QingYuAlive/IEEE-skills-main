# IEEE Transactions Observations

This legacy filename is retained only for compatibility with existing resource maps. The active content defines IEEE Transactions figure behavior.

## Observed Review Expectations

- Reviewers expect exact protocols, not visual implication.
- Reviewers challenge unsupported OOD and robustness claims first.
- Reviewers look for missing ablations of proposed loss terms, temperatures, sampling policies, and architectural components.
- Reviewers penalize inconsistent notation between figures, method equations, and captions.
- Reviewers expect recent baselines and uncertainty across repeats.

## Practical Implications

- Put split names directly in axes, legends, or captions.
- Show each claimed mechanism as a measurable visual contrast.
- Pair qualitative feature plots with quantitative compactness metrics.
- State complexity when the figure implies efficiency.
- Use the same method ordering in all panels and tables.
