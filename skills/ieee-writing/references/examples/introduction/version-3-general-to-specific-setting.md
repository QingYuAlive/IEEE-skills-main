# Introduction Example: General Task To Specific Setting

Use this pattern when the general field is broad but the manuscript studies a precise target protocol.

```text
Time-series classification maps multichannel temporal observations to task labels under constraints imposed by sampling rate, sensor placement, and nonstationary operating conditions. This paper focuses on the specific setting of cross-force sEMG gesture recognition, where the model is trained on labeled low- and medium-force windows and evaluated on a held-out high-force target domain.
```

The first sentence names the general task and constraints. The second sentence narrows to the exact source-target protocol.
