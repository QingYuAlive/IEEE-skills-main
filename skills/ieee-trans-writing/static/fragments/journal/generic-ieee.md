# Journal: Generic IEEE Transactions

Use this fragment when the user does not name TCYB, THMS, or TNNLS.

## Drafting Priorities

- Maintain the IEEE Transactions sequence: formal problem, notation, method equations, protocol, results, ablations, complexity, and limitations.
- Prefer precise technical verbs over promotional claims.
- Use section titles and paragraph openings that help reviewers locate task, objective, protocol, and evidence quickly.
- Keep the first paragraph technical. State the engineering task and challenge before application motivation.

## Venue-Neutral Checks

- Abstract contains problem, limitation, mechanism, and evidence.
- Introduction contributions are categorized and evidence-facing.
- Method is equation-first and reproducible.
- Experiments define OOD protocols before claiming generalization.
- Discussion explains mechanisms and failure modes without restating tables.
