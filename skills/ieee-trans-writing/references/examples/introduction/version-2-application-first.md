# Introduction Example: Application First With Immediate Formalization

Use this pattern only when the application is familiar to the target reviewers and the first paragraph still becomes technical within one sentence.

```text
Reliable myoelectric control requires gesture decoders that remain accurate when users change contraction force, electrode contact, or recording session. We study this requirement as an OOD classification problem in which labeled source domains are available during training and the target force domain is held out until evaluation.
```

The application sentence is allowed because the second sentence immediately defines the OOD learning setting.
