# Journal: Generic IEEE Transactions

Use this fragment when the user does not name TCYB, THMS, or TNNLS.

## Polishing Priorities

- Keep prose objective, compact, and technically dense.
- Preserve equations, symbols, assumptions, and protocol details.
- Bind claims to datasets, splits, metrics, baselines, ablations, and limitations.
- Prefer reviewer-defense clarity over stylistic smoothness.

## Checks

- The first mention of each symbol is defined.
- The draft distinguishes domain, dataset, split, distribution, subject, session, and force level.
- Robustness and generalization claims are tied to explicit OOD protocols.
- Method prose contains mappings, objectives, and inference-path information.
