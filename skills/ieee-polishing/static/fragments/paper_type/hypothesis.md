# Paper Type: Hypothesis Or Theory-Guided

## IEEE Reader Questions

- What hypothesis, assumption, theorem, or bound is being tested?
- Which conditions are required?
- Which experiments distinguish support from non-discriminating consistency?
- Which results would weaken or refute the explanation?

## Polishing Priorities

- Separate theorem conditions from empirical claims.
- Do not let a theoretical bound imply empirical dominance.
- Preserve assumptions and failure conditions.
- Use cautious language when evidence is consistent with but does not identify the mechanism.

## Argument Order

```text
assumption -> formal claim -> mechanism -> empirical test -> alternative explanation -> boundary
```
